## 1. IA (메뉴 구조)

```
로그인 (/login)
│
홈 대시보드 (/)                          ← 최근 리워드 지급 내역 위젯 포함
│
├── 1. 파트너사 관리 [마스터/외부팀]
│   ├── 1-1. 파트너사 목록/조회          /partners
│   └── 1-2. 파트너사 등록/수정          /partners/create  |  /partners/:id/edit
│
├── 2. 서비스 관리
│   ├── 2-1. 서비스 홈 (담당 서비스 선택)  /services
│   └── 2-2. 개별 서비스 대시보드         /services/:serviceId/*
│       ├── 2-2-1. 정보 수정              /services/:serviceId/info
│       ├── 2-2-2. 알림 관리              /services/:serviceId/notifications
│       ├── 2-2-3. 리워드 관리            /services/:serviceId/rewards
│       ├── 2-2-4. 오퍼 관리 [마스터/외부팀]  /services/:serviceId/offers
│       ├── 2-2-5. 수동 지급 [마스터/외부팀]  /services/:serviceId/provisions
│       ├── 2-2-6. CBO 등록하기 [마스터/외부팀]  /services/:serviceId/cbo
│       └── 2-2-7. 운영자 관리            /services/:serviceId/operators
│
├── 3. 서비스 관리 [마스터/외부팀]
│   └── 3-1. 전체 서비스 관리             /admin/services
│
└── 4. 관리자 페이지 [마스터 전용]
    ├── 4-1. 사용자 관리                  /admin/users
    └── 4-2. 진입 권한 부여               /admin/users/:id/edit (4-1 수정 내 포함)
```

## 2. 사용자 역할

| 역할 | 설명 | 접근 범위 |
|------|------|-----------|
| **마스터** | 최고 관리자 (카카오뱅크 내부) | 전체 메뉴, 관리자 페이지 포함 |
| **외부팀** | 서비스 담당 관리자 (카카오뱅크 내부) | 담당 서비스 한정, 전체 서비스 관리(담당만), 관리자 페이지 불가 |
| **제휴사** | 파트너사 담당자 (외부) | 소속 서비스 조회 전용, 생성/수정/삭제 불가 |

## 6. 용어 정리

| 용어 | 설명 |
|------|------|
| 파트너사 | 신사업팩토리와 제휴한 외부 회사 |
| 서비스 | 파트너사가 운영하는 개별 앱/서비스 |
| Service ID | 서비스 식별자 (제휴사영문3자_서비스약자3자_번호, e.g., SMP_SHO_001) |
| 오퍼 | 외부 API를 등록하고 리워드와 매핑한 단위 |
| 리워드 | 사용자에게 지급하는 보상 (쿠폰/현금/랜덤) |
| 수동 지급 | 운영 목적으로 특정 사용자에게 직접 리워드를 지급하는 기능 |
| CBO | Closed Beta Operation |
| 마스터 | 전체 권한을 가진 최고 관리자 계정 (카카오뱅크 내부) |
| 외부팀 | 카카오뱅크 내부 서비스 담당자 (구 "운영자") |
| 제휴사 | 파트너사 소속 외부 담당자 |
| 대표 운영자 | 서비스 운영/결재 책임자 |
| CI | 사용자를 식별하는 고유 식별자 (수동 지급 대상 지정 시 사용) |
