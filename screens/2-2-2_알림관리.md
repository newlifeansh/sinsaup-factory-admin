# 2-2-2 — 알림 관리 (v3.0)

> URL: `/services/:serviceId/notifications`
> 접근 역할: **마스터 / 외부팀** (편집) / **제휴사** (조회 전용)
> 최종 업데이트: 2026-04-01

---

## 화면 목표
특정 서비스의 사용자에게 보낼 알림(앱 푸시, 인앱 메시지 등)을 생성하고, 발송 스케줄을 관리합니다.

---

## A. 목록 화면

### 헤더 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 페이지 제목 | Text | "알림 관리" |
| + 알림 등록 | Button (Primary) | 클릭 시 등록 화면으로 이동 (마스터/외부팀만 노출) |

### 목록 테이블

| 컬럼명 | 컴포넌트 | 설명 |
|--------|----------|------|
| 알림 ID | Text | 시스템 자동 생성 고유 ID |
| 명칭 | Text | 알림 명칭 |
| 채널 | Badge | 알림탭 / 앱푸쉬 |
| 운영 기간 | Text | 시작일 ~ 종료일 |
| 발송 시간 | Text | 매일 발송 시간 (HH:MM) |
| 상태 | Toggle | 활성/비활성 전환 (마스터/외부팀만 조작 가능) |
| 등록일 | Text | YYYY.MM.DD |
| 관리 | Button | 수정 버튼 → 등록/수정 화면으로 이동 (마스터/외부팀만 노출) |

---

## B. 등록/수정 화면

> URL: `/services/:serviceId/notifications/create` | `/services/:serviceId/notifications/:id/edit`

### 페이지 제목
- "키워드 알람 등록" (등록 시) / "키워드 알람 수정" (수정 시)

### 기본 정보 영역

| 필드명 | 컴포넌트 | 필수 여부 | 설명 |
|--------|----------|-----------|------|
| 명칭 | Input | 필수 | 알림 식별 이름 |
| 설명 | TextArea | 선택 | 알림 내용 설명 |
| 활성화 여부 | Toggle | 필수 | 기본값: 활성 |
| 템플릿 ID | Input | 필수 | 발송 템플릿 식별자 |
| 메시지 채널 | Segmented (알림탭 / 앱푸쉬) | 필수 | 발송 채널 선택 |
| 서비스 | Select (readonly) | - | 현재 서비스 자동 설정 |

### 운영 기간 영역

| 필드명 | 컴포넌트 | 필수 여부 | 설명 |
|--------|----------|-----------|------|
| 시작 시간 | DatePicker | 필수 | 운영 시작일 |
| 종료 시간 | DatePicker | 필수 | 운영 종료일 |

### 스케줄 설정 영역

| 필드명 | 컴포넌트 | 필수 여부 | 설명 |
|--------|----------|-----------|------|
| 기준 시간 | TimePicker | 필수 | 매일 발송할 시간 (HH:MM) |

### 하단 버튼 영역

| 요소 | 컴포넌트 | 동작 |
|------|----------|------|
| 초기화 | Button (Default) | 폼 전체 초기화 |
| 알람 등록 / 수정 | Button (Primary) | 저장 API 호출 후 목록으로 이동 |

---

## 역할별 권한

| 역할 | 목록 조회 | 등록/수정 | Toggle 조작 |
|------|-----------|-----------|-------------|
| 마스터 | O | O | O |
| 외부팀 | O | O | O |
| 제휴사 | O | X (버튼 숨김) | X (Readonly) |

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| 상태 Toggle 변경 | 즉시 API 호출로 상태 변경 (확인 모달 없음 또는 간단한 confirm) |
| 저장 성공 | 목록으로 이동, 성공 Toast 표시 |
| 필수 필드 미입력 | 필드 하단 Validation 메시지 표시 |
| 수정 화면 진입 | 기존 알림 정보 폼에 pre-fill |
| 제휴사가 등록/수정 URL 직접 접근 | 403 또는 목록으로 리다이렉트 |
