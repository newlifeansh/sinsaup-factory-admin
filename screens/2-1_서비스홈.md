# 2-1 — 서비스 홈 (v3.0)

> URL: `/services`
> 접근 역할: 마스터 / 외부팀 / 제휴사
> 최종 업데이트: 2026-04-01

---

## 화면 목표
로그인한 사용자가 자신의 역할에 따라 접근 가능한 서비스 목록을 확인하고, 관리할 서비스를 선택하여 진입합니다.

---

## 화면 요소

### 헤더 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 페이지 제목 | Text | "서비스 관리" |

### 콘텐츠 영역 — 서비스 목록

| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 서비스 카드 목록 | Card Grid 또는 List | 접근 가능한 서비스를 카드/리스트 형태로 나열 |
| 서비스명 | Text | 서비스 이름 |
| 서비스 로고/아이콘 | Image | 서비스 대표 이미지 |
| 상태 뱃지 | Badge | 운영중 / 준비중 / 비활성 |

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| 서비스 카드 클릭 | `/services/:serviceId/info`로 이동 (개별 서비스 대시보드 진입) |
| 접근 가능 서비스 없음 | "접근 가능한 서비스가 없습니다. 마스터 관리자에게 권한을 요청하세요." 빈 상태 표시 |

---

## 역할별 노출 차이

| 역할 | 노출 범위 | 설명 |
|------|-----------|------|
| 마스터 | 전체 서비스 | 시스템에 등록된 모든 서비스 |
| 외부팀 | 담당 서비스 | 4-3에서 권한 부여된 서비스만 |
| 제휴사 | 소속 서비스 | 소속 제휴사에 연결된 서비스만 |
