# 2-2 — 개별 서비스 대시보드 (공통 레이아웃) (v3.0)

> URL: `/services/:serviceId/*`
> 접근 역할: 마스터 / 외부팀 / 제휴사 (해당 서비스 진입 권한 보유자)
> 최종 업데이트: 2026-04-01

---

## 화면 목표
선택된 특정 서비스에 대한 모든 관리 기능(정보, 알림, 리워드 등)을 수행하는 메인 페이지입니다.

---

## 공통 레이아웃 요소

### 상단 영역
| 요소 | 설명 |
|------|------|
| 현재 서비스명 | 상단에 관리 중인 서비스 이름 표시 (예: "스마트픽") |
| Breadcrumb | 신사업팩토리 > 개별 서비스 관리 |

### 사이드 서브 메뉴

| 메뉴 | URL | 화면 ID | 접근 역할 |
|------|-----|---------|-----------|
| 정보 수정 | `/services/:serviceId/info` | 2-2-1 | 마스터 / 외부팀 / 제휴사 |
| 알림 관리 | `/services/:serviceId/notifications` | 2-2-2 | 마스터 / 외부팀 / 제휴사 |
| 리워드 관리 | `/services/:serviceId/rewards` | 2-2-3 | 마스터 / 외부팀 / 제휴사 |
| 오퍼 관리 | `/services/:serviceId/offers` | 2-2-4 | 마스터 / 외부팀 (제휴사 숨김) |
| 수동 지급 | `/services/:serviceId/provisions` | 2-2-5 | 마스터 / 외부팀 (제휴사 숨김) |
| CBO 등록 | `/services/:serviceId/cbo` | 2-2-6 | 마스터 / 외부팀 / 제휴사 |
| 운영자 관리 | `/services/:serviceId/operators` | 2-2-7 | 마스터 / 외부팀 / 제휴사 |

> 사이드 메뉴 선택에 따라 오른쪽 콘텐츠 영역이 변경됨 (SPA 라우팅)
> 제휴사 역할인 경우 오퍼 관리, 수동 지급 메뉴가 사이드바에서 숨김 처리됨

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| 진입 권한 없는 서비스 URL 직접 접근 | 403 접근 거부 페이지 또는 `/services`로 리다이렉트 |
| serviceId 존재하지 않음 | 404 페이지 표시 |
| 서브 메뉴 항목 클릭 | 해당 URL로 이동, 선택 메뉴 하이라이트 |
| 제휴사가 숨김 메뉴 URL 직접 접근 | 403 접근 거부 |
