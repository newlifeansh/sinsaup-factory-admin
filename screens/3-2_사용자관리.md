# 3-2 — 사용자 관리 (마스터 전용)

> URL: `/admin/users`
> 접근 역할: **마스터 전용**

---

## 화면 목표
마스터 관리자가 이 어드민 시스템을 사용하는 모든 사용자(마스터/운영자) 계정을 관리합니다.

---

## 화면 요소

### 헤더 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 페이지 제목 | Text | "사용자 관리" |
| + 관리자 등록 | Button (Primary) | 신규 관리자 계정 생성 폼으로 이동 |
| 검색 | Input | 이름 / 이메일 키워드 검색 |

### 목록 테이블

| 컬럼명 | 설명 |
|--------|------|
| 관리자 ID | 시스템 자동 생성 고유 ID (예: ADM-001) |
| 이름 | 관리자 성명 |
| 이메일 | 로그인 이메일 |
| 담당 서비스 | 권한 부여된 서비스명 목록 (요약) |
| 최근 로그인 | 마지막 로그인 일시 (YYYY.MM.DD HH:MM) |
| 관리 | 수정 버튼 → `/admin/users/:id/edit` (3-3 기능 포함) |

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| 검색어 입력 | 이름 또는 이메일 포함 실시간 필터링 |
| 수정 버튼 클릭 | `/admin/users/:id/edit`으로 이동 (3-3 진입 권한 부여 포함) |
| + 관리자 등록 클릭 | 신규 관리자 등록 폼으로 이동 |
| 운영자가 URL 직접 접근 | 403 접근 거부 또는 홈으로 리다이렉트 |
