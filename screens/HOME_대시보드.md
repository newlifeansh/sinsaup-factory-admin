## 1. 요약 통계 카드 (역할별)

### 마스터 (4개)

| 카드 | 설명 |
|------|------|
| 전체 파트너사 수 | 등록된 파트너사 총 수 |
| 전체 서비스 수 | 등록된 서비스 총 수 |
| 활성 리워드 수 | 현재 활성 상태인 리워드 총 수 |
| 금일 리워드 지급 건수 | 오늘 지급된 리워드 건수 |

### 외부팀 (4개)

| 카드 | 설명 |
|------|------|
| 담당 서비스 수 | 본인이 담당하는 서비스 수 |
| 담당 파트너사 수 | 담당 서비스에 연결된 파트너사 수 |
| 활성 리워드 수 | 담당 서비스 내 활성 리워드 수 |
| 금일 리워드 지급 건수 | 담당 서비스의 오늘 지급 건수 |

### 제휴사 (3개)

| 카드 | 설명 |
|------|------|
| 내 서비스 수 | 소속 제휴사의 서비스 수 |
| 활성 리워드 수 | 내 서비스 내 활성 리워드 수 |
| 금일 리워드 지급 건수 | 내 서비스의 오늘 지급 건수 |
