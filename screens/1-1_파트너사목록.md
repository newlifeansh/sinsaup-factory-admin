# 1-1 — 파트너사 목록/조회

> URL: `/partners`
> 접근 역할: 마스터 / 운영자

---

## 화면 목표
등록된 모든 파트너사를 목록 형태로 조회하고, 검색 및 관리 기능(수정)을 제공합니다.

---

## 화면 요소

### 헤더 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 페이지 제목 | Text | "파트너사 관리" |
| + 파트너사 등록 | Button (Primary) | 클릭 시 `/partners/create`로 이동 |

### 검색 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 파트너사명 검색 | Input | 파트너사명 키워드 검색 |
| 검색 버튼 | Button | 검색 실행 |

### 목록 테이블

| 컬럼명 | 설명 | 정렬 |
|--------|------|------|
| 파트너사 ID | 시스템 자동 생성 고유 ID (예: PRT-001) | 좌측 |
| 파트너사 | 파트너사 이름 | 좌측 |
| 대표자 | 파트너사 대표자명 | 좌측 |
| 연동 서비스 | 해당 파트너사와 연동된 서비스 개수 | 중앙 |
| 가입일 | 파트너사 등록일 (YYYY.MM.DD) | 좌측 |
| 관리 | 수정 버튼 → `/partners/:id/edit`으로 이동 | 중앙 |

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| 검색어 입력 후 검색 | 파트너사명 포함 검색 결과 필터링 |
| 검색 결과 없음 | "검색 결과가 없습니다" 빈 상태 표시 |
| 수정 버튼 클릭 | 해당 파트너사 수정 페이지(`/partners/:id/edit`)로 이동 |
| + 파트너사 등록 클릭 | 파트너사 등록 페이지(`/partners/create`)로 이동 |
