# 2-2-4 — 오퍼 관리 (v3.0)

> URL: `/services/:serviceId/offers`
> 접근 역할: **마스터 / 외부팀 전용** (제휴사 접근 불가 — 사이드바에서 숨김)
> 최종 업데이트: 2026-04-01

---

## 화면 목표
외부 API를 '오퍼'로 등록하고, API 결과에 따라 지급할 '리워드'를 매핑(연결)합니다.

---

## 중요 정책

> **오퍼번호/상세일련번호/부점코드/텔러번호 등 내부 정보는 리워드 화면(2-2-3)에서 절대 노출 금지**
> 이 정보들은 오퍼 관리 화면 내에서만 조회/관리 가능

---

## A. 목록 화면

### 헤더 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 페이지 제목 | Text | "오퍼 관리" |
| + 오퍼 등록 | Button (Primary) | 클릭 시 등록 페이지로 이동 |

### 목록 테이블

| 컬럼명 | 컴포넌트 | 설명 |
|--------|----------|------|
| 오퍼 ID | Text | 시스템 자동 생성 고유 ID |
| 오퍼명 | Text | 오퍼 이름 |
| 연결 서비스 | Text | 연결된 서비스의 Service ID (svcId) |
| API 엔드포인트 | Text (truncate) | 연동된 API 엔드포인트 URL |
| 매핑된 리워드 | Text | 연결된 리워드명 목록 (요약) |
| 상태 | Toggle | 활성/비활성 전환 |
| 관리 | Button Group | 수정 버튼 / 테스트 버튼 |

---

## B. 등록/수정 페이지 (탭 구조)

> URL: `/services/:serviceId/offers/create` | `/services/:serviceId/offers/:id/edit`

### Tab 1 — API 정보

| 필드명 | 컴포넌트 | 필수 여부 | 설명 |
|--------|----------|-----------|------|
| 오퍼명 | Input | 필수 | 오퍼 식별 이름 |
| 연결 서비스 (svcId) | Select (Readonly) | - | 현재 서비스 자동 설정 (Service ID 표시) |
| API 엔드포인트 | Input (URL) | 필수 | 호출할 외부 API URL |
| HTTP Method | Select | 필수 | GET / POST / PUT 등 |
| Request Spec | Code Editor | 선택 | API Request 스펙 (JSON) |
| Response Spec | Code Editor | 선택 | API Response 스펙 (JSON) |

> 오퍼 등록 시 **연결 서비스(svcId)** 필드가 자동 설정되어 어떤 서비스에 속하는 오퍼인지 명확히 연결

### Tab 2 — 리워드 매핑

| 필드명 | 컴포넌트 | 필수 여부 | 설명 |
|--------|----------|-----------|------|
| 오퍼 조건 | TextArea | 필수 | 리워드 지급 트리거 조건 설명 |
| 매핑된 리워드 선택 | Select (검색 가능) | 필수 | 해당 서비스의 리워드 목록에서 선택 |
| + 조건 추가 | Button | - | 조건-리워드 매핑 행 추가 |

> 복수의 조건-리워드 매핑 가능 (행 추가/삭제)

### Tab 3 — 테스트

| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 테스트용 Request | Code Editor | 테스트 요청 데이터 직접 입력 |
| 테스트 실행 | Button (Primary) | API 호출 실행 |
| 실행 결과 | Code Editor (Read-only) | 응답 결과 표시 |

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| 탭 이동 | 입력 데이터 유지 |
| 테스트 실행 | Tab 1의 엔드포인트+Method+Request Spec 기반으로 API 호출, Tab 3 결과 영역에 표시 |
| 상태 Toggle 변경 | 즉시 API 호출로 상태 변경 |
| 저장 성공 | 목록으로 이동, 성공 Toast 표시 |
| 수정 페이지 진입 | 기존 오퍼 정보 탭별 pre-fill |
| 제휴사가 URL 직접 접근 | 403 접근 거부 또는 `/services`로 리다이렉트 |
