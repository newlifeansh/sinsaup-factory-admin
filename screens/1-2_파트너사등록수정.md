# 1-2 — 파트너사 등록/수정

> URL: `/partners/create` (등록) | `/partners/:id/edit` (수정)
> 접근 역할: 마스터 / 운영자

---

## 화면 목표
새로운 파트너사를 시스템에 등록하거나 기존 파트너사의 정보를 수정합니다.

---

## 화면 요소

### 헤더 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 페이지 제목 | Text | 등록 시 "파트너사 등록" / 수정 시 "파트너사 정보 수정" |

### 입력 폼

| 필드명 | 컴포넌트 | 필수 여부 | 설명 |
|--------|----------|-----------|------|
| 파트너사명 | Input | 필수 | 파트너사 공식 명칭 |
| 대표자명 | Input | 필수 | 파트너사 대표자 성명 |
| 사업자등록번호 | Input | 필수 | `000-00-00000` 형식 |
| (기타 파트너사 정보) | - | - | 추후 필요 필드 추가 예정 |

### 버튼 영역
| 요소 | 컴포넌트 | 동작 |
|------|----------|------|
| 저장 / 수정 | Button (Primary) | 등록: 저장 API 호출 후 `/partners`로 이동 / 수정: 수정 API 호출 후 `/partners`로 이동 |
| 취소 | Button (Default) | `/partners`로 이동 (변경 사항 미저장) |

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| 필수 필드 미입력 후 저장 | 필드 하단 Validation 메시지 표시 |
| 사업자등록번호 형식 오류 | 형식 유효성 에러 메시지 표시 |
| 저장 성공 | `/partners` 목록으로 이동, 성공 Toast 표시 |
| 수정 페이지 진입 | 기존 파트너사 정보 폼에 pre-fill |
