# 2-2-6 — CBO 등록 (v3.0) [NEW]

> URL: `/services/:serviceId/cbo`
> 접근 역할: **마스터 / 외부팀** (생성/수정) / **제휴사** (조회 전용)
> 최종 업데이트: 2026-04-01

---

## 화면 목표
서비스 정식 출시 전, 지정된 사용자를 대상으로 비공개 테스트(Closed Beta Operation) 환경을 생성하고 관리합니다. CBO 링크를 자동 생성하고 유효기간 및 사용자 권한을 관리합니다.

---

## A. 목록 화면

### 헤더 영역
| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 페이지 제목 | Text | "CBO 등록" |
| + CBO 등록 | Button (Primary) | 클릭 시 등록 화면으로 이동 (마스터/외부팀만 노출) |

### 목록 테이블

| 컬럼명 | 컴포넌트 | 설명 |
|--------|----------|------|
| CBO ID | Text | 시스템 자동 생성 고유 ID |
| CBO명 | Text | CBO 식별 이름 |
| CBO 링크 | Text (truncate) + Copy | 자동 생성된 CBO 링크 URL, 복사 버튼 |
| 유효기간 | Text | 시작일시 ~ 종료일시 |
| 상태 | Badge | 활성 / 만료 / 비활성 |
| 등록일 | Text | YYYY.MM.DD |
| 관리 | Button | 수정 버튼 → 수정 화면으로 이동 (마스터/외부팀만 노출) |

## B. 등록/수정 화면

> URL: `/services/:serviceId/cbo/create` | `/services/:serviceId/cbo/:id/edit`

### 기본 정보

| 필드명 | 컴포넌트 | 필수 여부 | 설명 |
|--------|----------|-----------|------|
| CBO명 | Input | 필수 | CBO 식별 이름 |
| CBO 설명 | TextArea | 선택 | CBO 상세 설명 |
| 유효기간 | DateTime RangePicker | 필수 | CBO 유효 시작일시 ~ 종료일시 |

### CBO 링크

| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| CBO 링크 | Input (Readonly) | `https://.../cbo/{serviceId}` 등록 완료 시 자동 생성 |
| 링크 복사 | Button | 클립보드 복사 |
| 링크 미리보기 | Button | 새 탭에서 링크 미리보기 |

> CBO 링크는 등록 완료 후 시스템이 자동 생성하며, 수정 불가

### 접근 허용 사용자 관리

| 요소 | 컴포넌트 | 설명 |
|------|----------|------|
| 사용자 유형 | Radio | 임직원 / 제휴사 사용자 |
| 사용자 정보 | Input | 임직원=ID/이름 검색(@kakaobank.com), 제휴사=고객번호 직접 입력 |
| + 추가 버튼 | Button | 하단 목록에 추가 |

### 허용된 사용자 목록

| 컬럼명 | 설명 |
|--------|------|
| No | 순번 |
| 유형 | Badge: 임직원 / 제휴사 |
| 정보 | ID 또는 고객번호 |
| 이름 | 사용자 성명 |
| 등록일 | YYYY.MM.DD |
| 관리 | 삭제 버튼 (마스터/외부팀만 노출) |

### 버튼 영역

| 요소 | 컴포넌트 | 동작 |
|------|----------|------|
| 저장 / 수정 | Button (Primary) | 저장 API 호출 후 목록으로 이동 |
| 취소 | Button (Default) | 목록으로 이동 (변경 미저장) |

## 역할별 권한

| 역할 | 목록 조회 | 등록/수정 | 사용자 관리 |
|------|-----------|-----------|-------------|
| 마스터 | O | O | O (추가/삭제) |
| 외부팀 | O | O | O (추가/삭제) |
| 제휴사 | O | X (버튼 숨김) | X (조회만, 저장/취소/추가/삭제 숨김) |

---

## 동작 정의

| 케이스 | 동작 |
|--------|------|
| CBO 등록 완료 | CBO 링크 자동 생성, 목록으로 이동, 성공 Toast |
| 유효기간 만료 | 상태 자동으로 "만료"로 변경 |
| 링크 복사 클릭 | 클립보드에 복사, 성공 Toast 표시 |
| 사용자 추가 | 유형 선택 후 정보 입력 → 목록에 추가 |
| 필수 필드 미입력 | 필드 하단 Validation 메시지 표시 |
| 제휴사가 등록/수정 URL 직접 접근 | 403 또는 목록으로 리다이렉트 |
